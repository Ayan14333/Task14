# Task14 - JavaScript Number Operations

## Project Description
This project demonstrates basic number-based programs using JavaScript. It is beginner-friendly and covers important logic-building concepts like loops, conditions, and mathematical operations.

## Features
The program performs the following operations:

✅ Sum of first n natural numbers
✅ Multiplication table of a number
✅ Check whether a number is Prime
✅ Find all Factors of a number
✅ Calculate Sum of Digits
✅ Check Armstrong Number

## Concepts Used
- Variables (let)
- Loops (for, while)
- Conditional statements (if-else)
- Operators (%, +, /)
- Math functions (Math.pow(), Math.floor())

## How to Run
- Open any code editor (VS Code recommended)
- Copy the code into a .js file

- Run using:

- node filename.js

- OR open browser console and paste the code
